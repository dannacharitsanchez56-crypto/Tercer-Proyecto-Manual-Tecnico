Los layouts en Astro son una de las herramientas más importantes para organizar un proyecto web de manera profesional.

Permiten crear una estructura base que se reutiliza en múltiples páginas, evitando repetir código.

---

## 🧠 ¿Qué es un layout?

Un layout es una plantilla general que define la estructura de una página web.

Normalmente incluye:
- Encabezado (header)
- Menú de navegación
- Contenido principal
- Pie de página (footer)

---

## 🎯 ¿Para qué sirven?

Los layouts sirven para:

- Reutilizar código en varias páginas
- Mantener el diseño consistente
- Reducir errores y duplicación
- Mejorar la organización del proyecto
