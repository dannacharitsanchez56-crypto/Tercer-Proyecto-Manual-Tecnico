Bienvenidos al Manual

Astro permite trabajar con archivos Markdown (.md), lo que facilita la creación de contenido como blogs, manuales o documentación técnica.

Markdown es un lenguaje ligero que permite escribir contenido estructurado de forma sencilla.

🧠 ¿Qué es Markdown?

Markdown es un formato de texto que permite escribir contenido usando símbolos simples.

Ejemplo:

  • título

  • subtítulo

  • negrita
    • listas

🚀 ¿Por qué usar Markdown en Astro?

Astro soporta Markdown de forma nativa, lo que permite:

  • Crear páginas rápidas
  • Escribir contenido sin HTML complejo
  • Organizar blogs o documentación
  • Separar contenido del diseño

📄 Ejemplo básico de archivo Markdown

# Hola mundo

Este es un archivo Markdown dentro de Astro.

## Subtítulo

- Punto 1
- Punto 2
- Punto 3
