Bienvenidos al Manual

El despliegue es el proceso de publicar tu sitio web en internet para que otras personas puedan verlo. En Astro, este proceso es muy sencillo porque el framework genera sitios estáticos optimizados.

🧠 ¿Qué es el despliegue?

Desplegar significa:

  • Subir tu proyecto a internet
  • Hacerlo accesible desde un enlace
  • Publicarlo en una plataforma

🚀 Paso 1: Crear la versión de producción

Antes de subir el proyecto debes compilarlo:

npm run build
