Bienvenidos al Manual

Cuando creas un proyecto en Astro, el sistema genera una estructura de carpetas organizada para mantener el código limpio, escalable y fácil de entender.

Entender esta estructura es fundamental para poder desarrollar correctamente cualquier sitio web.

🧠 ¿Qué es la estructura de un proyecto?

La estructura del proyecto es la forma en que se organizan los archivos y carpetas dentro de Astro. Cada carpeta tiene una función específica dentro del sistema.

📁 Estructura general

Cuando creas un proyecto con Astro, verás algo como esto:

src/
  pages/
  components/
  layouts/
public/
astro.config.mjs
package.json
README.md
