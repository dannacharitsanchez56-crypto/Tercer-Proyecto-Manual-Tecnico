Bienvenidos al Manual

Antes de comenzar con Astro es importante contar con ciertos conocimientos y herramientas instaladas en el sistema operativo.

Requisitos básicos

  • Conocimiento básico de terminal (Linux o Windows PowerShell)
  • Conocimientos de HTML y CSS
  • Nociones básicas de JavaScript

Herramientas necesarias

  1. Node.js

    • Permite ejecutar JavaScript fuera del navegador
    • Se recomienda versión LTS

  2. npm

    • Gestor de paquetes de Node.js

  3. Editor de código

    • Visual Studio Code recomendado

Sistemas operativos compatibles

  • Linux (Ubuntu, Debian, etc.)
  • Windows 10 o superior
  • macOS

Tener estos elementos correctamente instalados es fundamental para evitar errores durante la creación del proyecto.
