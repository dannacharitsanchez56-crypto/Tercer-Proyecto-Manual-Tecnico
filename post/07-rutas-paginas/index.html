Bienvenidos al Manual

Uno de los conceptos más importantes en Astro es el sistema de rutas basado en archivos.

Este sistema permite crear páginas web simplemente agregando archivos dentro de una carpeta, sin necesidad de configurar rutas manualmente.

🧠 ¿Qué es una ruta?

Una ruta es la dirección que se escribe en el navegador para acceder a una página web.

Ejemplo:

  • /about
  • /contact
  • /blog En Astro, estas rutas se crean automáticamente.

📁 ¿Cómo funciona en Astro?

Astro utiliza la carpeta:

src/pages/
