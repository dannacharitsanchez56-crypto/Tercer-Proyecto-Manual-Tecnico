Bienvenidos al Manual

Node.js es un entorno de ejecución que permite correr JavaScript en el servidor. Astro depende de Node.js para funcionar correctamente.

Instalación en Windows

  1. Ir a la página oficial: https://nodejs.org
  2. Descargar la versión LTS
  3. Ejecutar el instalador
  4. Verificar instalación:
node -v
npm -v
