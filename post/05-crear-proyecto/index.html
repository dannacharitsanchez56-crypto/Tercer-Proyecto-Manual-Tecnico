Bienvenidos al Manual

Crear un proyecto en Astro es el punto de inicio para construir un sitio web moderno, rápido y organizado. Astro permite crear proyectos listos para desarrollo en pocos pasos usando su herramienta oficial CLI.

🧠 ¿Qué es un proyecto en Astro?

Un proyecto en Astro es una estructura de carpetas y archivos donde se desarrolla toda la aplicación web. Dentro de él puedes encontrar:

  • Páginas web (pages)
  • Componentes reutilizables
  • Layouts o plantillas
  • Archivos estáticos como imágenes

🚀 Paso 1: Crear el proyecto

Abre la terminal en la carpeta donde quieres crear tu proyecto y ejecuta:

npm create astro@latest
