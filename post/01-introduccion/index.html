Bienvenidos al Manual

Este manual tiene como objetivo explicar paso a paso cómo desarrollar un sitio web estático multi página utilizando el framework Astro junto con Tailwind CSS.

Astro es un framework moderno que permite crear sitios web rápidos, ligeros y basados en componentes. A diferencia de otros frameworks, Astro envía muy poco JavaScript al navegador, lo que mejora el rendimiento.

En este documento se abordarán conceptos fundamentales como:

  • Instalación del entorno en Linux y Windows
  • Creación de proyectos con Astro
  • Uso de componentes
  • Manejo de layouts
  • Implementación de Tailwind CSS

El objetivo final es construir un sitio tipo “manual técnico” que funcione como documentación profesional y práctica.
